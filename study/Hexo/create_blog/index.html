使用Github Pages+Hexo搭建简单博客

发表于 2019-01-14 | 分类于 学习 ， Hexo | 浏览

Github Pages本是用于管理介绍Projects的静态页面，不过由于其空间免费稳定，可用于搭建个人博客。
Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。Hexo支持Markdown，以及一键部署，对普通用户来说，非常简单便捷

三 安装Hexo

1 安装Hexo

点击鼠标右键，选中Git Bash Here，进入Git Bash，运行

2 Hexo初始化

建立相关目录，如：E:\Hexo_test，在此目录中打开Git Bash（进入此目录，点击右键，选择Git Bash Here）
执行hexo init

3 在本地打开hexo

在E:\Hexo_test中打开cmd或powershell，执行与hexo g 后再执行hexo server
结果如下：

至此本地Hexo搭建完成

4 安装hexo-deployer-git

为了后续能将Hexo页面顺利push到Git Pages，需安装hexo-deployer-git
在E:\Hexo_test中打开cmd或powershell，执行npm install hexo-deployer-git –save

虽然结果有报错，但是对后续貌似没什么影响，暂时忽略
